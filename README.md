# 旅游推荐系统 python899

本项目为一款基于Python开发的旅游信息采集与推荐系统，运用了数据爬虫、协同过滤算法以及数据可视化技术。

## 技术栈

- **编程语言**：Python
- **开发框架**：Django
- **数据库**：MySQL
- **网络爬虫技术**：requests
- **机器学习算法**：协同过滤算法
- **前端技术**：BootStrap
- **数据可视化**：ECharts

## 功能模块

- **数据采集**：利用网络爬虫技术采集旅游相关数据。
- **数据分析**：对采集到的数据进行处理分析。
- **推荐算法**：运用基于用户的协同过滤推荐算法，为用户推荐旅游信息。
- **数据可视化**：通过ECharts将分析结果以图表形式直观展现。

## 系统角色

- **用户**：访问系统，获取个性化旅游推荐。
- **管理员**：负责系统维护和数据分析。

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x

## 部署步骤

1. 环境准备：安装Python、MySQL、Django等必备环境。
2. 数据库配置：根据项目需要设置MySQL数据库。
3. 项目部署：运行项目代码，完成部署。
